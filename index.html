FAQ

Straight answers. No hype.

Do I need new software or a new CRM?

No. We can start with what you already use (email + Sheets), or set up a simple CRM if needed.

Is this “AI”? Do my customers see AI?

We focus on outcomes (faster replies and better follow-up). Customers simply get faster responses and clearer questions.

Will this replace my admin?

It reduces admin workload. Most clients keep human control for final quotes, but automation handles the repetitive parts.

How do you handle GDPR and data?

We only collect what you already collect for quoting. We can keep data in your existing tools (Google/Microsoft) and follow best practices.

What if I want to change the messages later?

You’ll have editable templates and a clear process. Optional monthly support is available for ongoing improvements.
